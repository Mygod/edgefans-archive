上传 EdgeMod

Chinese, Simplified
  • 登录或注册后发表评论

在制作完成你的 EdgeMod 后你很可能希望把你的 EdgeMod 上传到 EDGE Fans 上，很不幸，这种玩意儿就现在而言是完全可能的。

  1. 只需轻轻点击页面左方的“添加内容”，然后点击 Mod。（快捷入口）
  2. 在 Title 处输入你的 EdgeMod 的标题。
  3. 在下方的 Description 处输入你对你的 EdgeMod 的描述，如果想要插入些截图，可以将页面向下滚动些，我在那里写了些亲切的提示。
  4. 在 Downloads 处选择文件后点击上传，上传你的 EdgeMod！
  5. 点击保存提交！

如果你觉得你的路径不够美观，你可以在下方的 URL路径设置 中取消选中 Generate automatic URL alias，然后在下方的 URL别名 内输入您的 mod 路径。如 mods/mygod/starry。为了避免你的 mod 被删除，请以 mods 打头，中间空格用 － 代替。
此外虽然不强制，但是推荐你的 edgemod 开放源代码，即你修改过的文件，来帮助其他开发者。你可以到 Ubuntu Pastebin、GitHub 或自己的网盘等上传然后在你的 mod 页面复制粘贴一个链接。
